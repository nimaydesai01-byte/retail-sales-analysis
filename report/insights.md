# Retail Sales Analysis

## Project Overview
This project analyzes a retail sales dataset using SQL to identify sales trends and business insights.

## Key Insights

1. The West region generated the highest overall revenue compared to other regions.

2. Technology products contributed the highest share of total sales.

3. Sales show strong increases during certain months indicating seasonal demand.

4. Sub-categories such as Phones and Chairs appear among the top-selling products.

## Tools Used

- SQL
- Microsoft Excel
- XAMPP (MySQL)
- Visual Studio Code
