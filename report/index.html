Retail Sales Data Analysis

SQL + Excel Data Analysis Project

Project Overview

This project analyzes a retail sales dataset using SQL queries and Excel visualizations to discover sales trends, regional performance, and top-selling products. The goal is to extract meaningful business insights that can help decision making.

Dataset Summary

  • Dataset: Superstore Retail Sales
  • Total Records: ~10,000+
  • Fields Analyzed: Region, Category, Sub-Category, Sales, Order Date
  • Analysis Performed Using SQL Queries

Tools Used

  • SQL – Data querying and analysis
  • Microsoft Excel – Data visualization
  • XAMPP / MySQL – Database management
  • Visual Studio Code – Project development

Key Insights

  • West region generated the highest revenue among all regions.
  • Technology category contributed the largest share of total sales.
  • Sales increase during certain months indicating seasonal trends.
  • Phones and Chairs are among the top performing product sub-categories.
